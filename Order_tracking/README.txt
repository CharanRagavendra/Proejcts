This folder contains the files for Order Tracking project.
 
This project is based on TEXT to SQL using Langchain and OpenAPI. Here we use Agents and tools.

SQLite was used for storing information.

"DB" folder contains files regarding
-> Create, Delete and update of the tables (ord1 and ord2 are the two tables present)
-> Table data

StreamLit is used for creating a user interface, where the user can ask questions.

"run.py" and "run2.py" are the main files for execution.
"run2.py" has better user friendly answer for the questions.
