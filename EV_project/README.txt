This folder contains the files for Electric Vehicle charging stations project.
 
This project is based on TEXT to SQL using Langchain and OpenAPI.

Mysql local command client was used to store information about 
->Area, 
->Locations of that area where the charging station is located, 
->Number of ports availabe currently in that station,
->price to charge per hour,
->Contact Number of the person incharge of that station,
->id which is used uniquely identify the station.

StreamLit is used for creating a user interface, where the user can ask questions.

"run.py" is the main file for execution
